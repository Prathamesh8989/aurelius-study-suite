# 4. CORE INTELLIGENT ENGINE

Include JSDoc documentation for every algorithm.

---

Spaced Repetition Engine

When a topic is completed, schedule recall sessions using a simplified forgetting curve:

+1 day
+7 days
+30 days

Store recall sessions in a Recall Schedule list.

Display them in a timeline.

---

GPA Gap Analyzer

Implement:

Performance Delta =
(Target Grade - Current Grade) * Difficulty / Days Until Exam

Purpose
Prioritize subjects where improvement is most critical.

Display

Priority heatmap
Urgency ranking list.

---

Focus Efficiency Metric

Focus Efficiency =
Completed Pomodoros / Planned Pomodoros

Display as a percentage stat.
